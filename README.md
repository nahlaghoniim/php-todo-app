# Simple Todo App

A lightweight **PHP-based Todo application** that demonstrates reading and writing JSON data, handling forms, and performing CRUD operations. Perfect as a foundation project for beginners.

---

## Features

- **Read & Write JSON**: Stores todos in a `todo.json` file.
- **Add Todos**: Users can add new todos using a simple form.
- **Mark as Completed**: Toggle the completion status of a todo.
- **Delete Todos**: Remove todos from the list.
- **CRUD Functionality**: Create, Read, Update, Delete operations are all handled.
